Gmail Phishing Detector

✅
Extension is active

This extension highlights potentially dangerous emails in Gmail and provides security information.

How it works:

1. Emails with potential phishing risks are highlighted with a red border

2. Click the "⚠ Why?" button to see why an email was flagged

3. Use "More Info" to chat with the security assistant

4. Use "Unflag" to mark the email as safe and hide future warnings

Note:
This is a simulation using mock data for demonstration purposes. No real email analysis is performed.
